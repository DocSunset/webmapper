A web-based GUI for the mapper protocol.

To run:
	1. Copy _mapper.so and mapper.py from libmapper/swig into webmapper directory.
	2. Run webmapper.py from terminal
		- Terminal will display "serving at port #####" (where ##### is a string of 5 numbers)
	3. Open a browser
	4. Type "localhost:#####" into the address bar (where ##### is the same string of 5 numbers displayed in terminal.
